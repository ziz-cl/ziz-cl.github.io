# QuickRequest - 프로젝트 설명서

## 📋 프로젝트 개요

**QuickRequest**는 개발자를 위한 Chrome 확장프로그램으로, HTTP API 요청을 빠르게 생성, 저장, 실행할 수 있는 도구입니다.

### 핵심 컨셉
- **빠른 API 테스트**: Postman보다 가볍고 빠르게 API 요청 테스트
- **사이트별 관리**: 각 웹사이트별로 요청을 자동 분류
- **배치 실행**: 하나의 파라미터에 여러 값을 적용하여 대량 요청 실행
- **쿠키/헤더 자동 포함**: 현재 사이트의 인증 정보 자동 사용

---

## 🎯 주요 기능

### 1. HTTP 요청 관리
- GET, POST, PUT, DELETE, PATCH 메서드 지원
- JSON 형식의 파라미터 및 Body 데이터
- 요청별 이름 지정 및 저장

### 2. 사이트별 자동 분류
- 현재 접속한 사이트 URL을 기준으로 요청 자동 분류
- "모든 사이트에 적용" 옵션으로 전역 요청 설정 가능
- 사이트 전환 시 해당 사이트 요청만 표시

### 3. 배치 실행 모드 ⭐ (차별화 기능)
- 하나의 파라미터 키 지정 (예: `userId`)
- 여러 값을 줄바꿈으로 입력 (예: 123, 456, 789)
- 각 값마다 자동으로 요청 생성 및 순차 실행
- 결과를 테이블 형식으로 한눈에 확인
- 각 결과의 성공/실패 상태 표시
- 상세 응답 데이터 펼침/접힘 기능

### 4. 쿠키 및 헤더 관리
- 현재 사이트의 쿠키 자동 포함
- 기본 헤더(Content-Type, Accept) 자동 설정
- 인증이 필요한 API도 간편하게 테스트

### 5. 실시간 결과 확인
- 요청 실행 후 즉시 응답 확인
- HTTP 상태 코드 표시
- JSON 응답 자동 포맷팅
- 에러 메시지 상세 표시

## 🎨 UI/UX 특징

### 레이아웃
- **팝업 크기**: 450px × 400px (최소)
- **색상 스키마**:
  - 주 색상: `#0066cc` (파란색)
  - 성공: `#28a745` (초록색)
  - 에러: `#dc3545` (빨간색)
  - 배치 모드: `#17a2b8` (청록색)

### 탭 구조
1. **저장된 요청 탭**: 현재 사이트의 요청 목록
2. **새 요청 탭**: 새로운 요청 생성 폼

### 요청 카드 디자인
```
┌─────────────────────────────────────┐
│ [GET] 사용자 정보 조회  [모든 사이트] │
│ https://api.example.com/users       │
│                                     │
│ [실행] [편집] [삭제]                │
└─────────────────────────────────────┘
```

배치 요청일 경우:
```
┌─────────────────────────────────────┐
│ [GET] 다중 사용자 조회  [배치]      │
│ https://api.example.com/users       │
│ 배치 파라미터: userId               │
│                                     │
│ [배치 실행] [편집] [삭제]           │
└─────────────────────────────────────┘
```

### 배치 실행 결과 테이블
```
┌────────────────────────────────────────────────────┐
│ No │ 값    │ 상태        │ 응답 요약        │      │
├────┼───────┼─────────────┼──────────────────┼──────┤
│ 1  │ 123   │ ✓ 200 OK   │ {id, name, ...}  │ 상세 │
│ 2  │ 456   │ ✓ 200 OK   │ {id, name, ...}  │ 상세 │
│ 3  │ 789   │ ✗ 404      │ Not Found        │ 상세 │
└────────────────────────────────────────────────────┘
```

## 💻 기술 스택

### Chrome Extension
- **Manifest V3** (최신 버전)
- **Service Worker** (background.js)
- **Chrome Storage API** (로컬 데이터 저장)
- **Chrome Cookies API** (쿠키 접근)
- **Fetch API** (HTTP 요청)

### 프론트엔드
- Vanilla JavaScript (프레임워크 없음)
- HTML5
- CSS3 (Grid, Flexbox)

### 파일 구조
```
QuickRequest/
├── manifest.json          # 확장프로그램 설정
├── popup.html            # 팝업 UI

├── styles.css            # 스타일시트 (601줄)

│   ├── icon16.png
```

## 🎨 디자인 요구사항

### 1. 아이콘 디자인 (가장 중요!)

**필요한 아이콘 크기:**
- 16x16px (브라우저 툴바)
- 48x48px (확장프로그램 관리 페이지)
- 128x128px (Chrome Web Store)

**아이콘 컨셉:**
- HTTP 요청을 상징하는 디자인
- 화살표 또는 네트워크 통신을 나타내는 요소
- 간결하고 현대적인 느낌
- 파란색 계열 (#0066cc 기반)
- 16px에서도 식별 가능해야 함

**아이콘 아이디어:**
1. 오른쪽 화살표 + 'Q' 문자
2. 네트워크 노드 연결
3. API 엔드포인트 심볼
4. Thunder/Lightning bolt (빠름을 상징)
5. 문서 + 화살표 조합

### 2. 스크린샷 (Chrome Web Store용)

**필요한 스크린샷:**
- **크기**: 1280x800 또는 640x400
- **최소 1개, 권장 5개**

**촬영해야 할 화면:**
1. **메인 화면**: 저장된 요청 목록
2. **새 요청 생성**: 폼 입력 화면
3. **배치 실행 모달**: 값 입력 화면
4. **배치 결과 테이블**: 성공/실패 결과 표시
5. **응답 결과**: JSON 응답 표시

**스크린샷 가이드:**
- 밝은 배경에서 촬영
- 실제 사용 예시 데이터 포함
- 주요 기능이 보이도록
- 깔끔하고 정돈된 느낌

### 3. 프로모션 이미지 (선택사항)

**Small Tile**: 440x280
**Large Tile**: 920x680 (추천!)
**Marquee**: 1400x560

**포함할 요소:**
- 로고/아이콘

- 핵심 기능 3가지 강조
  - "빠른 API 테스트"
  - "배치 실행"
  - "사이트별 관리"

## 📊 사용 시나리오

### 시나리오 1: 개발자의 API 디버깅
```
개발자가 로컬 서버(localhost:3000)에서 작업 중
→ QuickRequest로 여러 API 엔드포인트 요청 저장
→ 각 요청을 빠르게 실행하며 응답 확인
→ 파라미터 수정하며 다양한 케이스 테스트
```

### 시나리오 2: QA의 대량 데이터 테스트
```
여러 사용자 ID로 동일한 API 테스트 필요
→ 배치 모드로 요청 생성
→ 사용자 ID 100개를 줄바꿈으로 입력
→ 한 번에 실행하여 결과 확인
→ 어떤 ID에서 에러가 나는지 한눈에 파악
```

### 시나리오 3: 블로그 사이트 API 테스트
```
블로그 사이트에 로그인된 상태
→ QuickRequest로 "댓글 작성" API 요청 생성
→ 현재 사이트 쿠키 사용 체크
→ 실행하면 로그인된 사용자로 댓글 작성
→ 인증 토큰 복사/붙여넣기 불필요
```

## 🚀 경쟁 우위

### vs Postman
- ✅ 브라우저에 통합되어 빠른 접근
- ✅ 현재 사이트 쿠키/헤더 자동 사용
- ✅ 사이트별 자동 분류
- ✅ 배치 실행 기능
- ✅ 가볍고 빠름

### vs Thunder Client (VS Code)
- ✅ 브라우저 컨텍스트에서 바로 실행
- ✅ 실제 브라우저 쿠키 사용
- ✅ 별도 앱 전환 불필요

### 독자적 기능
- **배치 실행 모드**: 다른 도구에 없는 차별화 기능
- **사이트별 자동 관리**: 컨텍스트 기반 요청 분류
- **원클릭 실행**: 저장된 요청 즉시 실행
